void AD1_OnEnd ( void  )

A handler that reads the last ADC measurement.

This handler accumulates the subsequent readings from the ADC until the maximum number of samples is achieved. Thereafter, an average is calculated and a flag is set, indicating that a new temperature measurement is ready to be published.
