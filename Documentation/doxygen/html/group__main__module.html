Initializing the communication (actually a minimalist version of it) and dc motor modules.

Infinite loop that checks if a message was received from the Terminal. If it was, then the msg is parsed and handled.

NOTE: This code is only used to test the dc_motor library.

Checks if a message was received. The flag 'message_received' is set when a sequence of "\r\n" arrives at UART0 RX (check UART0Events.c to more on that).

"P": PWM setting

This function is responsible for ensuring that the PWM passed as argument is valid

"D": Direction setting

"S": Stops the motor

"C": Continues to spin the motor with the last PWM setted
